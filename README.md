### ⚠️注意事项
1.本项目是vibe coding项目，主包是小白，所以未对mdx文件进行任何修改，仅修改css及js代码

2.本项目仅在`欧路词典win版`、`GoldenDict-ng win版`、`欧路词典Android版`上测试，其他软件兼容性未知

3.本项目不包含mdd发音文件以及发音图标，默认隐藏发音图标。若有相关需求，请自行下载mdd和图标，并在p.css文件中删除.pf的样式代码
```
.pf {
    display: none;
}
```
